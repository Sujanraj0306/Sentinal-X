### 🚀 Core Features
* **Automated Case Extraction:** Drag-and-drop FIRs/Evidence for instant OCR and JSON structuring.
* **The Debate Arena:** Multi-agent autonomous debate analyzing case facts in real-time.
* **Legal Document Generator:** 1-click generation of Petitions, Bail Applications, Legal Notices, and Contracts directly to PDF.
* **Integrated LLM Search:** Built-in legal research library.

### 💻 Tech Stack
* **Frontend:** React, Electron
* **Backend:** Node.js, Python
* **Database:** SQLite (VectorDB)
* **AI/LLM:** Google Gemini AI

### 🛠️ Quick Start / Installation

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   cd MADsystem
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```
   *(Note: Ensure you have followed the internal `.env` setup guide to place your Gemini API keys in the root directory before running)*

3. **Boot the application:**
   ```bash
   npm start
   ```
